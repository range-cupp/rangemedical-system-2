# Tighten this draft

Use this prompt when you have a draft tip that's too long, too soft, too generic, or doesn't quite land. Drop into the Claude project that has `CLAUDE.md` loaded.

---

## Prompt

Here's a draft daily tip. Tighten it per `CLAUDE.md`.

```
[paste your draft here — subject + body]
```

**What I want you to do:**
1. **Cut everything that isn't earning its place.** Adjectives, adverbs, hedges, throat-clears. Especially the second half of any sentence that starts with "and" or "but" — usually the second half is the only half worth keeping.
2. **Voice-check every line.** Anything that sounds like a wellness coach or a SaaS landing page → rewrite or kill.
3. **Make the action line concrete.** "Try cutting back" → "Last drink before 7pm." "Consider magnesium" → "400mg magnesium glycinate, 1 hr before bed."
4. **Punch up the subject.** 4-7 words. Sarcastic if it earns it.
5. **Final word count must be ≤180. Target ~120.** Read it out loud. If it takes more than 45 seconds → cut more.

**Output:**
- The tightened version, in the locked email format from `CLAUDE.md`.
- A one-line summary of what you cut and why (max 30 words).
- If anything in the original is unsalvageable, say so directly. Don't pretend bad ideas are good ones.

Go.

---

## Tips for using this prompt

- If you wrote it and it's already <140 words, tighten anyway. Best drafts get sharper, not bloated.
- If Claude's tightened version sounds different from your voice, push back. "Sound more like me. Less polished. Drop the alliteration."
- Keep the original next to the tightened. Sometimes the original opener was better — let yourself frankenstein the two.
